# TimerApp
This is a very simple timer manager app made in C++ with Qt. 

## Features
The following features are included. I conssidered them mandatory before releasing any binaries:
- Adding and removing timers inside the application
- Saving timers to disk and loading them from it
- Notifying the user via desktop notification when a timer finishes

## Future plans
- Settings menu to change the location timers are saved at
- Optionally playing a sound when a timer finishes
- Better UI for adding timers (having HH:MM:SS instead of seconds, at the very least)
- Cleaner UI as a whole
